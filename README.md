## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Sharan112212/Portfolio_Sharan_M.git
```

Move into the project directory:

```bash
cd Portfolio_Sharan_M
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
